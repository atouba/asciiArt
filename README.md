## ASCII Art

Ascii-art is a program that receives a string as an argument and prints it out as big ASCII art letters.

#### How to use: 

Command:
```bash
go run . "Hello"
```
Output:
```
 _    _          _   _          
| |  | |        | | | |         
| |__| |   ___  | | | |   ___   
|  __  |  / _ \ | | | |  / _ \  
| |  | | |  __/ | | | | | (_) | 
|_|  |_|  \___| |_| |_|  \___/  
                                
                                
```

Command:
```bash
go run . "Enjoy some\nASCII-art\!"
```
